# Procon-fightstick
AVRマイコンをNintendo Switch Pro Controllerに偽装するプログラムです

atmega32u4にて動作確認(UNOは未検証)

配線はPokeControllerやNX Macro Controllerと変わりません

必ず本体の設定にある「プロコントローラーの有線通信を有効にする」項目をonにしてから使用してください
